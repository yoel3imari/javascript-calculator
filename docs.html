JavaScript-Super-Calculator by yoel3imari

I created a scientific calculator using javascript

This calculator has these features:

  1. It allows you to do simple maths
  2. It records the history of operations you did
  3. It gives you the ability to calculate complexe operations that include special functions
  4. It allows you to tap numbers and operant using your keyboard

Rules to follow when using this calculator

To use this calculator properly, you need to follow these rules:

  1. All functions (Ln, e, cos, sin, ...) should be used with brackets to wrap their parameters. Ex: ln(5) is corret, ln5 is not
  2. Before using Yx you should tap a single '(' then write the base then tap Yx or X2 on the option section then write the power and close the brackets. Ex: (15+2)^(3) =
  3. Always make sure all brackets have an openning and a closing pair. Otherwise you will get a beautiful Error
  4. If you get "Error 2" that means the easult is "Infinity". It happens for an instance when you try to calculate Ln(0)or division by 0
